```plaintext
Necessário instalar a biblioteca networkx pyvis:
pip install networkx pyvis

Depois de rodar o código pela primeira vez, será gerado automaticamente uma pasta lib/ , um arquivo medicina_grafo.html e vai te redirecionar a um navegador para visualizar o grafo.
Caso não seja redirecionado, pesquise pelo caminho do arquivo medicina_grafo.html no seu navegador. Exemplo: file:///C:/Users/Casa/Desktop/-/Davi/Teoria_dos_Grafos/PI/medicina_grafo.html

Estrutura:
PI/
│── imagens/    # Pasta com as imagens dos médicos
│── grafo.py    # Código principal (NetworkX + PyVis)
